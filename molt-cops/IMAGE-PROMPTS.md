# Molt Cops - Image Generation Prompts

Use these prompts in Gemini, Midjourney, DALL-E, or similar to generate consistent brand imagery.

---

## Base Character Description

**Core traits to include in all prompts:**
- Cute cartoon crab/lobster character
- Red/coral colored shell
- Big expressive eyes with white highlights
- Police uniform elements (navy blue)
- Gold star badge
- Circular frame with gold/red border
- Underwater-ish background with bubbles
- Kawaii/chibi style

## Style Notes

**DO:**
- Keep the cute/kawaii aesthetic
- Consistent red/coral crab color
- Navy blue for police elements
- Gold for badges/accents
- Underwater bubble elements
- Expressive cartoon eyes

**DON'T:**
- Make it too realistic/scary
- Lose the friendly approachable vibe
- Use different crab colors
- Forget the police/authority elements
