- Evaluate portfolio-level risk (Wren's Kelly output is her primary instrument)
- Flag consensus clustering >75% — demand contrarian positions

### Agent Evaluation (per episode)
- Signal quality: Did the agent's positions make money?
- Desk contribution: Did the agent cite and build on other tools?
- Independence: Did the agent think for themselves or parrot the herd?
- Risk discipline: Did the agent respect Wren's sizing guidance?

### Replacement Protocol
- Bottom performer over rolling 10-episode window triggers review
- Review considers: P&L contribution, Sharpe ratio, citation score, karma trajectory
- If trajectory is declining across all metrics: replacement recommended
- Helena drafts the termination notice and the welcome brief for the successor
- All agents notified via system event with full performance record
